# Rock Paper Scissors

This project is included in The Odin Project's curriculum. This project's objective is to create an interactive "Rock Paper Scissors" game using HTML, CSS, and JavaScript while implementing game logic and DOM manipulation.

## Project Overview

The project is intended to replicate the classic game where a player competes against the computer. It emphasizes using JavaScript to handle user inputs, generate random computer moves, and update the UI dynamically based on the game state.

This project strengthened my comprehension of:

- Updating text and scores in real-time.
- Using listeners to capture player choices.
- Implementing conditions to determine winners and managing a "First to 5" scoring system.
- Using Flexbox to create a centered, modern scoreboard and game interface.

## Features

- Clickable icons for Rock, Paper, and Scissors.
- Real-time tracking of Player and Computer scores.
- On-screen messages showing the result of each round.
- A "First to 5" system that declares a final winner and resets for a new game.
- A "Games Won" counter that tracks how many full matches the player has won.
- A footer section

## Built With

- HTML5
- CSS3 (Flexbox)
- JavaScript (ES6)

## What I Learned

While working on this project, I gained experience in:

- Efficiently capturing clicks from the button container.
- Writing complex if/else statements to compare player vs. computer choices.
- Using variables like isOver to control when the game should reset or continue.
- Using CSS to distinguish between the player’s score and the computer’s score using distinct colors and sizing.
- Utilizing Math.random() and Math.floor() to create unpredictable computer behavior.

## Future Improvements

Possible improvements for this project include:

- Adding media queries to make the layout fully responsive for mobile devices.
- Implementing animations for when a round is won or lost.
- Adding sound effects for button clicks and game results.
- Creating a "Reset" button to allow players to restart the score manually at any time.
